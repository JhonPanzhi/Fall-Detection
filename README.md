# FallDetection
A desktop application that works with Hikvision webcam to detect falls in elderly people living alone. 一款检测老人跌倒并自动联系子女的桌面应用程序

## 功能
1. 自动检测老人跌倒
2. 可以全天候检测
3. 可以自动发送邮件联系亲人救援

## 所需设备
海康威视DS-IPC-E42H-IWPT

## 界面功能
1. 联系人编辑
2. 系统监护日志
3. 实时显示原始图像和系统监护图像
4. 带有电子说明书

## 相关技术
1. OpenCV
2. 海康威视SDK
3. MFC
